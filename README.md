Employee Management System (C#)

A simple **console-based Employee Management System** developed in **C#** using **Object-Oriented Programming (OOP)** concepts.

## Features
- Add Employee and Manager
- View all employees
- Update employee salary
- Delete employee
- Manager-specific details
- Menu-driven console application
  
## OOP Concepts Used
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

## Technologies
- C#
- .NET Console Application
- Visual Studio
- Git & GitHub
